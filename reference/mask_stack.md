# Sample Points in Raster Stack

Masks stack to a circular area of interest.

## Usage

``` r
mask_stack(stack)
```

## Arguments

- stack:

  ([terra::SpatRaster](https://rspatial.github.io/terra/reference/SpatRaster-class.html))  
  Raster stack.

## Value

[terra::SpatRaster](https://rspatial.github.io/terra/reference/SpatRaster-class.html)
