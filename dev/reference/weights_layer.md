# Weights Layer Generator

Generates a weights layer when passed to
[`generate_stack()`](https://mlr3spatial.mlr-org.com/dev/reference/generate_stack.md).

## Usage

``` r
weights_layer(id, in_memory = FALSE)
```

## Arguments

- id:

  (`character(1)`)  
  Layer id.

- in_memory:

  (`logical(1)`)  
  If `FALSE` (default), layer is written to disk.
